# 消息模块代码地图

**最后更新:** 2026-03-09
**入口点:** src/matrix/send.ts, src/matrix/send-queue.ts

## 架构

```
src/matrix/
├── send.ts                 # 消息发送核心
├── send-queue.ts           # 发送队列
├── send/
│   ├── markdown.ts         # Markdown 转换
│   ├── format.ts           # 消息格式化
│   ├── poll.ts             # 投票消息
│   ├── reaction.ts         # 反应消息
│   └── media.ts            # 媒体上传
├── format.ts               # 通用格式化工具
└── poll-types.ts           # 投票类型定义
```

## 模块说明

### 发送层模块

| 模块 | 用途 | 关键功能 |
|------|------|----------|
| `send.ts` | 消息发送核心 | 统一发送接口、处理各类消息类型、错误重试 |
| `send-queue.ts` | 发送队列 | 异步消息队列、流量控制、批量发送 |
| `send/markdown.ts` | Markdown 转换 | 将 Markdown 转换为 Matrix 支持的 HTML 格式 |
| `send/format.ts` | 消息格式化 | 消息体构建、元数据设置、格式转换 |
| `send/poll.ts` | 投票处理 | 创建投票消息、格式化投票选项、处理投票响应 |
| `send/reaction.ts` | 反应处理 | 发送消息反应、取消反应 |
| `send/media.ts` | 媒体上传 | 上传文件到 Matrix 媒体服务器、处理媒体元数据 |
| `format.ts` | 通用格式化 | 消息ID规范化、用户ID格式化、内容清理 |
| `poll-types.ts` | 投票类型 | 投票事件类型定义、投票状态管理 |

## 消息发送流程

```
出站消息 → 格式转换 → 类型路由（文本/媒体/投票/反应）→ 内容构建 →
发送队列 → 流量控制 → Matrix API 调用 → 响应处理 → 状态更新
```

## 支持的消息类型

| 消息类型 | 处理模块 | 说明 |
|----------|----------|------|
| 文本消息 | `send/format.ts` | 支持 Markdown 格式 |
| Markdown 消息 | `send/markdown.ts` | 转换为 Matrix 富文本格式 |
| 投票消息 | `send/poll.ts` | 支持单选/多选投票 |
| 反应消息 | `send/reaction.ts` | 表情反应 |
| 图片/文件 | `send/media.ts` | 支持图片、文档、音频、视频 |
| 线程回复 | `send/format.ts` | 支持在线程中回复 |
| 编辑消息 | `send.ts` | 支持编辑已发送消息 |

## 队列特性

- **异步发送**：非阻塞发送，不阻塞主流程
- **流量控制**：限制发送速率，避免触发 Matrix 服务器限流
- **重试机制**：失败自动重试，指数退避
- **批量处理**：多条消息批量发送
- **优先级**：支持消息优先级设置

## 格式转换

### Markdown 支持
- 标题、列表、链接
- 粗体、斜体、删除线
- 代码块、行内代码
- 引用、分割线
- 表格

### 媒体处理
- 自动检测文件类型
- 自动生成缩略图（图片）
- 提取元数据（音频/视频）
- 支持文件大小限制配置

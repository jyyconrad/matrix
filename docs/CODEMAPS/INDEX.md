# Matrix 插件代码地图索引

**最后更新:** 2026-03-09
**项目:** @openclaw/matrix
**版本:** 2026.3.7

## 概览

这是 OpenClaw 的 Matrix 渠道插件，实现了与 Matrix 协议的集成，支持即时通讯、群组聊天、线程消息、投票和媒体传输等功能。

## 架构层级

```
┌─────────────────────────────────────────┐
│              插件入口 (index.ts)        │
├─────────────────────────────────────────┤
│          渠道核心 (channel.ts)          │
├─────────┬─────────┬─────────┬─────────┤
│ 消息处理 │ 监控层  │ 发送层  │  工具   │
├─────────┼─────────┼─────────┼─────────┤
│  Matrix SDK 客户端 (matrix-bot-sdk)    │
└─────────────────────────────────────────┘
```

## 关键特性

- 🔌 完整的 Matrix 协议支持
- 👥 私聊、群组、线程消息
- 📊 投票功能
- 🎭 消息反应
- 📎 媒体文件传输
- 🔒 端到端加密支持
- 📋 多账户管理
- 🛡️ 安全策略和访问控制

## 入口点

- `index.ts` - 插件主入口
- `src/channel.ts` - 渠道插件核心实现
- `src/matrix/index.ts` - Matrix 客户端入口
- `src/matrix/monitor/index.ts` - 消息监控入口
